# 接口文档

**方式**：http

**域名**：api.inaction.fun

> 若无特殊说明，请求和返回数据都是 json 格式

## 注册

**url：**/register

**方法：** post

**参数：**

json格式

| key      | 类型   | 说明 |
| -------- | ------ | ---- |
| account  | string | 账号 |
| password | string | 密码 |
| name     | string | 名称 |

**返回数据：**

| 状态码 | 数据                 | 说明       |
| ------ | -------------------- | ---------- |
| 200    | {"msg":"ok"}         | 成功       |
| 409    | {"msg":"账号已存在"} | 账号已存在 |

## 登录

**url：**/login

**方法：** post

**参数：**

json格式

| key      | 类型   | 说明 |
| -------- | ------ | ---- |
| account  | string | 账号 |
| password | string | 密码 |

**返回数据：**

| 状态码 | 数据                 | 说明       |
| ------ | -------------------- | ---------- |
| 200    | {"msg":"ok"}         | 成功       |
| 404    | {"msg":"账号不存在"} | 账号不存在 |
| 409    | {"msg":"密码错误"}   | 密码错误   |
